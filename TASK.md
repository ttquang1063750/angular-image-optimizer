Mục tiêu: Cho phép người dùng so sánh chất lượng ảnh Gốc và ảnh Nén thông qua một giao diện slider (thanh trượt) trực quan.

## 📋 Danh sách công việc
- [x] **Component:** Thêm trạng thái `comparingFile` để quản lý file đang được chọn để so sánh.
- [x] **Giao diện (Item):** Thêm nút "So sánh" (biểu tượng 👁️) vào mỗi file item đã hoàn thành.
- [x] **Giao diện (Modal):** Xây dựng Modal so sánh với thanh trượt slider (Side-by-Side).
- [x] **Logic:** Xử lý kéo trượt slider để thay đổi độ rộng hiển thị của ảnh nén đè lên ảnh gốc.
- [x] **Kiểm thử:** Đảm bảo modal mở/đóng đúng và hiển thị đúng URL ảnh.
- [x] **Xác nhận:** Chạy `npm run lint`, `npm test` và `npm run build`.
